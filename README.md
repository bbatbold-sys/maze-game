# Maze Game

A single-file HTML maze game that randomly selects a level variation each time you play. Navigate through procedurally generated mazes with unique mechanics like darkness, ice physics, portals, breakable walls, and more.

## Features (Planned)
- Procedural maze generation
- Random level variation selection
- Multiple maze mechanics (dark, ice, portals, keys, gravity, mirror controls, breakable walls)
- Clean UI with level info and controls

## How to Play
Open `index.html` in any modern browser. Use arrow keys to navigate the maze.
